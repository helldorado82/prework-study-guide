HTML

  • The head element contains information used by the breowser to render the page
  • The body element contains the visible part of the show
  • Each element nested inside another is called a child
  • An element wrapping a child or children elements is called a parent
  • Lists can be ordered (with each item being numbered) or unordered (bulleted list)
  • Every item in the list needs to be wrapped by the "li"" tag
