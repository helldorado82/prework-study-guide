Prework Study Guide

HTML

  • The head element contains information used by the breowser to render the page
  • The body element contains the visible part of the show
  • Each element nested inside another is called a child
  • An element wrapping a child or children elements is called a parent
  • Lists can be ordered (with each item being numbered) or unordered (bulleted list)
  • Every item in the list needs to be wrapped by the "li"" tag

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Width and height indicate the space that the element will take inside their container
  • The selector * indicates all the elements in a page
  • Colour values can be defined either in plain text (blue, yellow, white, etc.) or in hex # (hexadecimal code)

Git

  • git status returns the branch we're currently working on
  • git checkout -b [name_of_the_branch] creates a new branch and moves to it
